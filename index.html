Como Usar:

  • Posicione sua mão diante da câmera para iniciar o controle
  • Mova a mão horizontalmente para controlar esquerda/direita
  • Mova a mão verticalmente para controlar cima/baixo
  • Aproxime/afaste a mão para controlar frente/trás
  • Junte polegar e indicador para fechar a garra
  • Pressione 'Q' no teclado para encerrar
Movimentos Ajustados:
→ Subir a mão = Braço DESCE
→ Descer a mão = Braço SOBE
→ Aproximar a mão = Vai para FRENTE
→ Afastar a mão = Vai para TRÁS

🔄 Controle Vertical

Movimento invertido: Quando a mão sobe, o braço desce. Quando a mão desce, o braço sobe.

↑ mão ↓ = ↓ braço ↑

🔄 Controle Profundidade

Movimento ajustado: Aproxime a mão da câmera para mover o braço para frente. Afaste para mover para trás.

⇨ mão ⇦ = ⇨ braço ⇦

✊ Controle da Garra

Junte a ponta do polegar e do indicador para fechar a garra. Afaste-os para abrir.
